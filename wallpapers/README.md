# Wallpapers

I use the package `imagemagick`, to generate the wallpaper with the correct format.

Example with `jjk_s3.jpg`:
`magick jjk_s3.jpg -background "#bb1f2c" -gravity center -extent 2880x1800 jjk_s3_2880x1800.jpg`

Example with `steel_haze.png`:
`magick steel_haze.png -background "#23283b" -gravity center -extent 2880x1800 steel_haze_2880x1800.jpg`
